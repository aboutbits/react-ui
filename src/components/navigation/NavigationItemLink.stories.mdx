# NavigationItemLink

The NavigationItemLink can be placed inside a menu for in-app routing.
It will make use of the specified LinkComponent in the ReactUIProvider.

export const Template = ({ ...args }) => (
  <NavigationItemLink
    {...args}
    href="#"
    content="Navigation Link"
    icon={IconAccountCircle}
  />
)

<Canvas>
  <Story name="Default">{Template.bind({})}</Story>
</Canvas>

### Props

<ArgsTable story="Default" />

### Theme

- `navigation.item.base`: Define the base style of the link, that is shared between different states.
- `navigation.item.normal`: Define the normal style of the link.
- `navigation.item.active`: Define the active style of the link.
- `navigation.item.icon.base`: Define the size of the icon.
- `navigation.item.content.base`: Define the floating direction of the content.
