# Breadcrumbs

This components render breadcrumbs, which can be used as a secondary navigation that helps to see the users location.

Typically, you use the `Breadcrumb.Link` component for the parent links and the `Breadcrumb.Text` component for the current page.
You can use the `BreadcrumbLoading` component or the `Breadcrumb.Loading` components to show a loading state.

export const Template = (args) => (
  <Breadcrumbs {...args}>
    <Breadcrumb.Link href="#">Start</Breadcrumb.Link>
    <Breadcrumb.Link href="#">User</Breadcrumb.Link>
    <Breadcrumb.Text>Settings</Breadcrumb.Text>
  </Breadcrumbs>
)
