# SectionFooter

The section footer is component that should be placed at the bottom of a `SectionContainer`.
It is used as a container for the section footer where, for example, the pagination can be placed.

export const Template = ({ children, ...args }) => (
  <SectionFooter {...args}>{children}</SectionFooter>
)
