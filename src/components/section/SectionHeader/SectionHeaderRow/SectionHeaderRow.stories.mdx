# SectionHeaderRow

This component is a container for elements inside a `SectionHeader`.
It manages the layout of the elements inside it. Choose between `stretch`, `spaceBetween`, `start`, `center` or `end`. It defaults to `spaceBetween`. `stretch` should only be used for a single element inside the container.

export const Template = ({ children, ...args }) => (
  <SectionHeaderRow {...args}>{children}</SectionHeaderRow>
)
