# SectionHeader

This component defines the header container of a section.

If your header contains several `SectionHeaderRow`s with a search field in the last row, you might consider to set the `increasedBottomSpacing` prop for improved spacing between header and content.

export const Template = ({ children, ...args }) => (
  <SectionHeader {...args}>{children}</SectionHeader>
)
