# SectionHeader

This component defines the header container of a section.

If your header contains or is breaking to serveral rows with a search field in the last row, you might consider to add a `mb-2` for improved spacing between header and content.

export const Template = ({ children, ...args }) => (
  <SectionHeader {...args}>{children}</SectionHeader>
)
