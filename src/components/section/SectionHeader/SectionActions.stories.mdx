# SectionActions

To create a section with actions, use the `SectionActions` component within `SectionHeader`, and add `SectionAction`s.

export const Template = ({ ...args }) => (
  <SectionActions>
    <SectionAction {...args} onClick={action('clicked')} />
  </SectionActions>
)

<Canvas>
  <Story
    name="Default"
    args={{ icon: IconFilterList, label: 'Filter', disabled: false }}

  </Story>
</Canvas>
