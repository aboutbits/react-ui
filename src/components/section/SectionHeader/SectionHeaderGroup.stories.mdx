# SectionHeaderGroup

This component is a container for elements inside a SectionHeader or SectionHeaderRow and manages the spacings between the elements.

export const Template = ({ children, ...args }) => (
  <SectionHeaderGroup {...args}>{children}</SectionHeaderGroup>
)

### Theme

- `section.headerGroup.base`: Define the base styles for the header group.
- `section.headerGroup.spacing.sm`: Define the styles for small spacing.
- `section.headerGroup.spacing.md`: Define the styles for medium spacing.
