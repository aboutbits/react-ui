# SectionContent

This component adds a div element and applies basic padding to it.
It is used as the container of the section content.

export const Template = ({ children, ...args }) => (
  <SectionContent {...args}>{children}</SectionContent>
)

<ArgsTable story="Default" />

### Theme

- `section.content.base`: Define the styles of the section content.
- `section.content.layout.oneColumnGrid`: The default theme applies a vertical gap between the items in this SectionContent.
- `section.content.layout.twoColumnGrid`: The default theme applies a grid styling to it. On screen sizes below `xl` it will have one column and on larger screens two.
