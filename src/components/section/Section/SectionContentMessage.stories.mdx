# SectionContentMessage

This component is used to show a message in a [SectionContent](/docs/components-section-sectioncontent--default-story).

It uses the [ContentMessage](/docs/components-content-contentmessage--default-story) for the message.

export const Template = ({ ...args }) => <SectionContentMessage {...args} />
