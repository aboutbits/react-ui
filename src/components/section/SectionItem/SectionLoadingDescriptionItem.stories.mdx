### Theme

- `section.loading.descriptionItem.base`: Define the style of the section description item
- `section.loading.descriptionItem.upper.base`: Define the padding, and the width for the upper loading bar.
- `section.loading.descriptionItem.lower.base`: Define the padding, and the width for the lower loading bar.
