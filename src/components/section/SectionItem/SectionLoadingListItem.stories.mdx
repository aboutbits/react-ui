# SectionLoadingListItem

This component can be used as placeholders for items during the fetch process. It is mainly used together with [SectionContenListLoading](/docs/components-section-sectioncontentlistloading--default-story) to generate a placeholder for lists during the fetch process.

### Theme

- `section.loading.listItem.base`: Define the style for the section list item
- `section.loading.listItem.start.base`: Define the padding, and the width for the start (left) loading bar
- `section.loading.listItem.end.base`: Define the padding, and the width for the end (right) loading bar.
