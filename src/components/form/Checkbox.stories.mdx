# Checkbox

This component can be used to create checkboxes. This input already has a label and handles error messages in `React Hook Form` forms.

It uses icons from the `@aboutbits/react-material-icons` library. To install:

```
npm i @aboutbits/react-material-icons
```

export const Template = (args) => <Checkbox {...args} name="accept-privacy" />

### Inside form

You can use the [FieldSet](/docs/components-form-fieldset--default-story) and the `applyInputHeight` prop to arrange the `Checkbox` properly inside a form layout.

export const TemplateInsideForm = () => (
  <FieldSet label="Privacy">
    <Checkbox
      name="chbeckbox-inside-form"
      label="Accept the privacy policy"
      applyInputHeight
    />
  </FieldSet>
)

<Canvas>
  <Story name="Inside form">{TemplateInsideForm.bind({})}</Story>
</Canvas>
