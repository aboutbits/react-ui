# ContentMessage

This component renders a message with icon and in given tone.
It can be used to create a warning or error message inside a section.

export const Template = (args) => <ContentMessage {...args} />

### Theme

- `content.message.base`: Define the basic style of the component.
- `content.message.contentContainer`: Define the style of the content container.
- `content.message.icon.base`: Define the style of the icon.
- `content.message.icon.tone[$tone]`: Define the style of the icon in the given tone.
- `content.message.iconContainer.base`: Define the style of the icon container.
- `content.message.iconContainer.tone[$tone]`: Define the style of the icon container in the given tone.
- `content.message.title.base`: Define the style of the title.
- `content.message.title.tone[$tone]`: Define the style of the title in the given tone.
- `content.message.text.base`: Define the style of the text.
- `content.message.text.tone[$tone]`: Define the style of the text in the given tone.
