# Dialog

This component is used to show an overlay dialog on the page. It supports different sizes and position behaviours on mobile and desktop.

This component uses the `Dialog` component from [HeadlessUI](https://headlessui.com/react/dialog).

### Subcomponents

You can use the following components for the different areas of the dialog (see examples below):

- Base components
  - `DialogHeaderArea`
  - `DialogHeaderRow`
  - `DialogHeaderGroup`
  - `DialogHeaderTitle`
  - `DialogContentArea`
  - `DialogFooterArea`
  - `DialogFooterActions`
- Convenience components
  - `DialogHeader`
  - `DialogHeaderLeftActionIcon`
  - `DialogHeaderCloseAction`
  - `DialogHeaderWithClose`
  - `DialogContent`
  - `DialogContentList`
  - `DialogFooterWithActions`

**Note:** The component does not include the button. The button is just used to activate the dialog changing the value of the prop `isOpen`.
