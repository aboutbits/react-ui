# Alert

This alert component can be used to highlight a message to the user.
It supports different types of tones.

export const Template = ({ children, ...args }) => (
  <Alert {...args}>{children}</Alert>
)

### Theme

- `alert.container.base`: Defines base styles of container
- `alert.container.tone[$tone]`: Defines custom styles per tone of the container
- `alert.content.base`: Defines base styles of content
- `alert.iconContainer.base`: Defines base styles of icon container
- `alert.iconContainer.tone[$tone]`: Defines custom styles per tone of the icon container
- `alert.icon.base`: Defines base styles of icon
- `alert.icon.tone[$tone]`: Defines custom styles per tone of the icon
