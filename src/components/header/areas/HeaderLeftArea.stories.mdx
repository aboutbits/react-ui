# HeaderLeftArea

This component is used in the header and represents the children on the left side of the header. For example it is used to show the back button to navigate to the previous screen.
It applies a margin to the right and displays the children in a row.

export const Template = ({ children, ...args }) => (
  <HeaderLeftArea {...args}>{children}</HeaderLeftArea>
)
