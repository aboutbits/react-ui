# HeaderBackAction

This component is used to add a `back action' on the left side of a header.

It leverages on the work of [ButtonIcon](/docs/components-button-buttonicon--docs), which adds the icon, and [HeaderLeftArea](/docs/components-header-headerleftarea--docs), which is in charge of the positioning.
