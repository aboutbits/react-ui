# HeaderRightActionIcon

This component is used to add a `ButtonIcon` to the right side of the header, e.g. to add a edit or search button.

It leverages on the work of [HeaderButtonIcon](/docs/components-header-headerbuttonicon--default-story), which adds a responsive `ButtonIcon`, and [HeaderRightArea](/docs/components-header-headerrightarea--default-story), which is in charge of the positioning.

export const Template = ({ ...args }) => (
  <HeaderRightActionIcon
    {...args}
    icon={Icon}
    onClick={actions('clicked')}
    label="Return to previous page"
  />
)
