# Breadcrumbs

This components render breadcrumbs, which can be used as a secondary navigation that helps to see the users location.

Typically, you use the `BreadcrumbLink` component for the parent links and the `BreadcrumbText` component for the current page.
You can use the `BreadcrumbLoading` component or the `BreadcrumbLoading` components to show a loading state.

export const Template = (args) => (
  <Breadcrumbs {...args}>
    <BreadcrumbLink href="#">Start</BreadcrumbLink>
    <BreadcrumbLink href="#">User</BreadcrumbLink>
    <BreadcrumbText>Settings</BreadcrumbText>
  </Breadcrumbs>
)
